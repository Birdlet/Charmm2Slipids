# Charmm2Slipids
### A simple Python script to convert non-protein from Charmm-gui output to Amber 
> This is also a Markdown test
***
## Supported chemicals
| Charmm | Amber | Name |
|--------|-------|------|
| POPC   | POPC  | 1-palmitoyl-2-oleoyl-sn-glycero-3-phosphocholine |
| CHL1   | CHOL  | Cholestrol |
| TI3P   | SOL   | Water |
| SOD    | NA    | Sodium |
| CHL    | CL    | Chlorine |
